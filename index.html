Mac Keyboard Layout - Esperanto

A set of super convenient keyboard layout schemata for the Esperanto language.

Installation

Just drop Esperanto.bundle into the directory ~/Library/Keyboard Layouts

~ is an alias for your home directory. (e.g. /Users/ada)

In Finder you can use the shortcut Command+Shift+G and then go to the directory by entering ~/Library/Keyboard Layouts.

If this is not your first time to install these keyboard layouts, remove the old layouts first. Next, logout and login again. Then reinstall.

Finally, use the system preference panel to add this layout. Enjoy! :-)

Features

  • Based on multiple layouts of the English Apple keyboards.

  • More latin letters from the combinations of accents and letters.

  • Support the x-system:

    c + x/X = ĉC + x/X = Ĉ
    g + x/X = ĝG + x/X = Ĝ
    h + x/X = ĥH + x/X = Ĥ
    j + x/X = ĵJ + x/X = Ĵ
    s + x/X = ŝS + x/X = Ŝ
    u + x/X = ŭU + x/X = Ŭ

  • Support the xx-system:

    c + x + x = cxxC + x + x = Cxx
    c + x + X = cxXC + x + X = CxX
    c + X + x = cXxC + X + x = CXx
    c + X + X = cXXC + X + X = CXX
    ...(similarly for g/h/j/s/u + x/X + x/X)...

  • Use ESC or Tab or irrelevant keys to cancel incomplete combinations:

    c + Tab = cC + Tab = C
    g + Tab = gG + Tab = G
    h + Tab = hH + Tab = H
    j + Tab = jJ + Tab = J
    s + Tab = sS + Tab = S
    u + Tab = uU + Tab = U
    c + x + Tab = ĉ(c + x + Tab ≠ cx)...
    S + u + g + a + r + c + o + a + t + i + n + g + . = Sugarcoating.
